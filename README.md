# insporation\*

insporation\* is a native [diaspora\*](https://diasporafoundation.org) client built with Flutter for iOS and Android. It is designed for everyday use on diaspora\* pods, including reading your stream, following conversations, and publishing posts from your phone.

diaspora\* is a decentralized social network: people sign up on independently run servers ("pods") and can still connect across the wider network. Its core principles are decentralization, freedom, and privacy.

## Compatibility

insporation\* is compatible with diaspora\* 0.9 and later.

## Build From Source

If you prefer to build locally:

1. Install [Flutter](https://flutter.dev).
2. Install Android SDK and/or Xcode.
3. Clone this repository.
4. Connect a device with developer mode enabled and run `flutter run`.
